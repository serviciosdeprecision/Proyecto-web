Este simulador utiliza funciones HTML5 que solo se encuentran en las últimas versiones de los navegadores y necesita mucha RAM. Si tiene problemas,verifique la compatibilidad.

Controles del teclado: 'z' para acercar, 'x' para alejar, 'n' para avanzar en la simulación.
Controles del mouse: haga clic con el botón izquierdo y arrastre para desplazarse (cuando se amplía).
